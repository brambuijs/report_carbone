Carbone Document Generator

Description

Carbone Document Generator is the ultimate solution for automating the creation of professional documents from your Odoo data and a Template file. Whether you need to generate invoices, contracts, reports, or any other type of document, Carbone makes it easy and efficient, without any coding. To get started: simply select your Odoo data, design a template in your preferred text editor (LibreOffice, Word, Excel, PowerPoint, or Google Docs), add Carbone tags (placeholders), and you’re ready to automate your documents!

This integration is the official Carbone.io connector for Odoo, developed and maintained by Mangono, an Official Carbone Partner ⭐️

Creating Carbone Templates

  • Step 1: Open Carbone Studio (the web interface for managing templates).
      • Step 2: On the home page, click "Start from scratch" to begin designing your template.
          • Step 3: Create your template using a text editor, and upload on Carbone Studio. The editor has three key sections to help you design your template:
            • 📄 Template Panel (on the top right panel): Where you upload and version your Template (e.g., invoices, contracts).
            • 📊 Data Model Editor (left panel): Here, you define the JSON data that fills your template (e.g., {d.firstName}).
            • 🔍 Document Preview (right panel): See a real-time PDF preview of your final document as you edit.

              Step 4: Once your design is ready, save your changes, and get a Template ID.
              • 1. Click "New document" to upload your template.
              • 2. Disable "Auto-reload" (to finalize changes).
              • 3. Click "Save": this generates a unique Template Version ID (you’ll need this for Odoo).

              Setting Up Document Generation in Odoo

              • In Odoo, go to Document Generation (main menu).
              • Click "New" to create a document workflow.

              Copy Your Template Version ID from Carbone into Odoo

              • 1. Paste your Template Version ID from Carbone Studio (1).
              • 2. Select the Odoo data model (e.g., Invoices, Contacts).
              • 3. Customize translation and language settings (if needed).
              • 4. Click "Add to Print Menu" (2), now you’re ready to generate documents!

Installation & Configuration

1. Installation

Install the module from the Odoo Store or directly from your Odoo interface:

  • Applications → Search for "Carbone Document Generator"
  • Click on "Install"

2. Configuration

Set up your Carbone.io API key:

  • Settings → Carbone.io Integration → API Keys
  • Enter your API key (Get yours at your Carbone Account)

3. Usage

To automate documents in Odoo:

  • Upload your templates to your Carbone account
  • Create a new "report" in the dedicated menu
  • Link them to Odoo models (invoices, quotes, etc.)
  • Generate your documents in one click!
